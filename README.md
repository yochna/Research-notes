# 📋 Research Notes Manager

Full-stack app deployed on Vercel with MongoDB Atlas.

**Stack:** React 18 · Vercel Serverless Functions · MongoDB Atlas

## Features
- Add / Edit / Delete notes
- Search notes live
- Filter by tag
- Pin notes to top
- Grid / List view
- Dark theme

## Deploy on Vercel
1. Push to GitHub
2. Import repo on vercel.com
3. Add environment variable: `MONGO_URI` = your Atlas connection string
4. Deploy ✅
